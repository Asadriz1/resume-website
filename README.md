# Resume Website

Personal resume website for Asad Rizvi showcasing education, skills, work experience, and contact information.

## Features

- 5 linked pages: Home, Education, Skills, Experience, Contact
- Modern, responsive design with CSS animations
- Professional styling with gradient backgrounds
- Hover effects and smooth transitions
- Mobile-friendly layout

## Deployment

This website is deployed using GitHub Pages.

### To Deploy:

1. Create a new repository on GitHub named `resume-website`
2. Initialize git and push:
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Resume website"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/resume-website.git
   git push -u origin main
   ```
3. Go to repository Settings > Pages
4. Select `main` branch and `/root` directory
5. Click Save
6. Your site will be available at: `https://YOUR_USERNAME.github.io/resume-website`

## Customization

- Replace `profile.jpg` with your actual profile picture (250x250px recommended)
- Update contact information in `contact.html`
- Modify colors in `styles.css` by changing CSS variables
